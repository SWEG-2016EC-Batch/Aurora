# Algorithm

## Psedocode
```

S1:START  

S2:DECLARE INTEGER variable: n  

S3:SET n = 6                     

S4:FOR i FROM 1 TO n DO  
     
S5:FOR j FROM 0 TO (n - i) DO  
      
S6:IF (i == 1 OR j == (n - i) OR j == 0) THEN  

            PRINT "*"  
            
        ELSE  
        
            PRINT " "  
            
 END IF  
 
 END FOR  

S7:PRINT newline  
    
END FOR  
 
S8:END
