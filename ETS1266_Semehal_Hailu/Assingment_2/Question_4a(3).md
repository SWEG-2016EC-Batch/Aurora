# Algorithm

## Psedocode

```
S1:Start  

S2:DECLARE INTEGER variables: row, colnums  

S3:PRINT "Enter the number of rows:" 

S4:READ row  

S5:PRINT "Enter the number of columns:"  

S5:READ colnums  


S6:FOR j FROM 1 TO colnums DO  
  
    SET row = row - (row - j)  
    
S7:FOR i FROM 1 TO row DO

        PRINT i
        
END FOR  

    PRINT newline            
END FOR
